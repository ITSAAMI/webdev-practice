📦 Topic 12: Block vs Inline Elements

Code Academy Nowshera | Instructor: Syed Amir Hayat | Batch: FS_B10

1. Block-Level Elements

Block elements ki 3 khaasiyaat hain:

  • 🔲 Full width lete hain (parent ki poori width)
  • ⬇️ Hamesha nayi line se shuru hote hain
  • 📏 Width/Height set kar sakte hain

Block Elements Ki List:

<div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <table>, <form>, <section>, <article>, <header>, <footer>, <nav>, <main>, <aside>, <blockquote>, <hr>

🖥️ Visual Demo:

<div> — Full width leta hai

<p> — Yeh bhi full width hai

<h3> — Heading bhi block hai

2. Inline Elements

Inline elements ki 3 khaasiyaat hain:

  • 📐 Sirf apne content jitna space lete hain
  • ➡️ Same line par rehte hain (nayi line nahi lete)
  • ❌ Width/Height set nahi kar sakte

Inline Elements Ki List:

<span>, <a>, <img>, <strong>, <em>, <b>, <i>, <u>, <br>, <input>, <label>, <abbr>, <code>, <small>, <sub>, <sup>

🖥️ Visual Demo:

Yeh ek <span> hai, aur yeh ek <a> hai, aur yeh ek <strong> hai — sab same line par hain!

3. Block vs Inline — Comparison Table

4. display: inline-block — Best of Both!

inline-block se element same line par rehta hai lekin width/height bhi set kar sakte hain:

<span style="display: inline-block; width: 100px; height: 100px; 
             background: red; color: white; text-align: center; 
             line-height: 100px; margin: 5px;">Box 1</span>

<span style="display: inline-block; width: 100px; height: 100px; 
             background: blue; color: white; text-align: center; 
             line-height: 100px; margin: 5px;">Box 2</span>

5. Nesting Rules

✅ Sahi — Block ke andar Inline:

<div>
  <p>Yeh <strong>sahi</strong> hai — block ke andar inline.</p>
</div>

❌ Galat — Inline ke andar Block:

<span>
  <div>Yeh GALAT hai — inline ke andar block!</div>
</span>

⚠️ Exception: <a> tag ke andar block elements rakh sakte hain (HTML5 mein allowed).
