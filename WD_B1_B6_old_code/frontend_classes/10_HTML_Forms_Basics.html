1. <form> Tag — Form Container

Form tag sab input fields ko wrap karta hai. action mein data kahan jayega aur method mein kaise jayega woh batate hain.

<form action="/submit" method="POST">
  <!-- Input fields yahan aayenge -->
</form>
AttributePurposeValues
actionData kahan bheja jayegaURL ya file path
methodData kaise bheja jayegaGET (URL mein) / POST (hidden)

💡 GET vs POST: GET se data URL mein dikhta hai (search ke liye). POST se data hidden rehta hai (login/signup ke liye).

2. Basic Input Types

📝 Code:

<form>
  <label for="name">Naam:</label>
  <input type="text" id="name" placeholder="Apna naam likhein" required>

  <label for="email">Email:</label>
  <input type="email" id="email" placeholder="example@gmail.com" required>

  <label for="pass">Password:</label>
  <input type="password" id="pass" placeholder="Password likhein" required>

  <label for="phone">Phone:</label>
  <input type="tel" id="phone" placeholder="03001234567">
</form>

4. <label> Tag — Input Ka Label

Label tag input ke saath connect hota hai for attribute se. Label click karne par input focus hota hai.

<!-- Method 1: for aur id match karein -->
<label for="username">Username:</label>
<input type="text" id="username">

<!-- Method 2: input ko label ke andar rakhein -->
<label>
  Username:
  <input type="text">
</label>

⚠️ Important: Hamesha <label> use karein — accessibility ke liye zaroori hai. Screen readers label padhte hain.

6. Submit aur Reset Buttons

<!-- Submit button — form data bhejta hai -->
<input type="submit" value="Submit Karein">

<!-- Reset button — sab fields clear karta hai -->
<input type="reset" value="Reset">

<!-- Button tag bhi use kar sakte hain -->
<button type="submit">Submit</button>
