📋 Topic 08: HTML Lists

Code Academy Nowshera | Instructor: Syed Amir Hayat | Batch: FS_B10

1. Unordered List — <ul>

Bullet points wali list — jab items ka order important nahi hota:

📝 Code:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>React</li>
</ul>

🖥️ Output:

  • HTML
  • CSS
  • JavaScript
  • React

2. Ordered List — <ol>

Numbered list — jab items ka order important ho:

📝 Code:

<ol>
  <li>HTML seekhein</li>
  <li>CSS seekhein</li>
  <li>JavaScript seekhein</li>
  <li>Projects banayein</li>
</ol>

🖥️ Output:

  1. HTML seekhein
  2. CSS seekhein
  3. JavaScript seekhein
  4. Projects banayein

OL Ke Special Attributes:

AttributeExampleResult
type="A"<ol type="A">A, B, C, D...
type="a"<ol type="a">a, b, c, d...
type="I"<ol type="I">I, II, III, IV...
type="i"<ol type="i">i, ii, iii, iv...
start="5"<ol start="5">5 se start hoga
reversed<ol reversed>Ulti counting (5, 4, 3...)

📝 Type Examples:

<ol type="A">
  <li>Islamabad</li>
  <li>Lahore</li>
  <li>Karachi</li>
</ol>

<ol type="I" start="3">
  <li>Third item</li>
  <li>Fourth item</li>
</ol>

🖥️ Output:

  1. Islamabad
  2. Lahore
  3. Karachi

  1. Third item
  2. Fourth item

3. Nested Lists — Lists Ke Andar Lists

📝 Code:

<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Backend
    <ul>
      <li>Node.js</li>
      <li>Python</li>
      <li>PHP</li>
    </ul>
  </li>
  <li>Database
    <ul>
      <li>MongoDB</li>
      <li>MySQL</li>
    </ul>
  </li>
</ul>

🖥️ Output:

  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python
    • PHP
  • Database
    • MongoDB
    • MySQL

4. Description List — <dl>, <dt>, <dd>

Term aur uski definition/description dikhane ke liye:

📝 Code:

<dl>
  <dt>HTML</dt>
  <dd>Web pages ka structure banane ki language</dd>

  <dt>CSS</dt>
  <dd>Web pages ko design aur style karne ki language</dd>

  <dt>JavaScript</dt>
  <dd>Web pages mein interactivity add karne ki language</dd>
</dl>

🖥️ Output:

HTML
Web pages ka structure banane ki language
CSS
Web pages ko design aur style karne ki language
JavaScript
Web pages mein interactivity add karne ki language

5. CSS Se List Styling

📝 list-style-type Values:

<ul style="list-style-type: disc;">   <!-- ● Default -->
<ul style="list-style-type: circle;"> <!-- ○ Khali gola -->
<ul style="list-style-type: square;"> <!-- ■ Square -->
<ul style="list-style-type: none;">  <!-- Koi bullet nahi -->

🖥️ Live Demos:

disc (default):

  • Item A
  • Item B

circle:

  • Item A
  • Item B

square:

  • Item A
  • Item B

none (no bullets):

  • Item A
  • Item B

6. Horizontal List — Navigation Menu

📝 Code:

<style>
  .nav-list { list-style: none; display: flex; gap: 20px; background: #0f3460; padding: 15px; border-radius: 8px; }
  .nav-list li a { color: #00d2ff; text-decoration: none; padding: 5px 10px; }
  .nav-list li a:hover { color: #e94560; }
</style>

<ul class="nav-list">
  <li><a href="#">Home</a></li>
  <li><a href="#">About</a></li>
  <li><a href="#">Courses</a></li>
  <li><a href="#">Contact</a></li>
</ul>

🖥️ Output:

  • Home
  • About
  • Courses
  • Contact

📝 Practice Tasks

  • Task 1: Ek recipe page banayein — ingredients <ul> mein aur cooking steps <ol> mein likhein. Nested list bhi use karein.
  • Task 2: HTML, CSS, aur JavaScript ke topics ki description list banayein (<dl>) jisme har topic ki definition ho.
  • Task 3: Horizontal navigation bar banayein jisme 5 links hon — CSS se styled aur hover effects ke saath.
