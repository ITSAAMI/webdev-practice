1. Semantic HTML Kya Hai?

Semantic ka matlab hai "meaningful" — semantic tags se browser, search engines, aur developers ko pata chalta hai ke content kya hai:

Non-Semantic (Purana)Semantic (Modern)Fark
<div id="header"><header>Tag ka naam hi batata hai ke yeh header hai
<div id="nav"><nav>Navigation links ka container
<div class="article"><article>Independent content piece

Semantic HTML Ke Faiday:

  • 🔍 SEO: Google content behtar samajhta hai
  • ♿ Accessibility: Screen readers page navigate kar sakte hain
  • 👨‍💻 Readability: Code padhna aasan hota hai
  • 🔧 Maintenance: Code maintain karna aasan

2. Important Semantic Tags

TagPurposeUse Case
<header>Page/section ka headerLogo, title, navigation
<nav>Navigation linksMenu, sidebar links
<main>Page ka main contentPrimary content (sirf EK baar)
<section>Thematic groupingAbout section, Contact section
<article>Independent contentBlog post, news article, comment
<aside>Side contentSidebar, related links, ads
<footer>Page/section ka footerCopyright, links, contact
<figure>Image + caption groupPhotos with descriptions
<figcaption>Figure ka captionImage description
<details>Collapsible contentFAQ, expandable sections
<summary>Details ka visible titleClick to expand
<time>Date/time markupPublished dates
<address>Contact informationAuthor/organization contact
<mark>Highlighted textSearch results highlighting

3. Complete Semantic Page Structure

<body>
  <header>
    <h1>Code Academy Nowshera</h1>
    <nav>
      <a href="#">Home</a> | <a href="#">Courses</a> | <a href="#">Contact</a>
    </nav>
  </header>

  <main>
    <section>
      <h2>Our Courses</h2>
      <article>
        <h3>HTML Course</h3>
        <p>Published: <time datetime="2024-01-15">Jan 15, 2024</time></p>
        <p>Complete HTML course for beginners...</p>
      </article>
      <article>
        <h3>CSS Course</h3>
        <p>Master CSS from scratch...</p>
      </article>
    </section>

    <aside>
      <h3>Quick Links</h3>
      <ul>
        <li><a href="#">Admissions</a></li>
        <li><a href="#">Timetable</a></li>
      </ul>
    </aside>
  </main>

    <address>
      Code Academy, GT Road, Nowshera, KPK<br>

</body>

4. <details> aur <summary> — Collapsible Content

Bina JavaScript ke collapsible/expandable section banayein:

<details>
  <summary>HTML kya hai? (Click to expand)</summary>
  <p>HTML ek markup language hai jo web pages ka structure define karti hai.</p>
</details>

🖥️ Live Demo (Click karein!):

❓ HTML kya hai?

HTML (HyperText Markup Language) ek markup language hai jo web pages ka structure define karti hai.

❓ CSS kya hai?

CSS (Cascading Style Sheets) se web pages ki design aur layout control ki jaati hai.

❓ JavaScript kya hai?

JavaScript ek programming language hai jo web pages mein interactivity add karti hai.

5. <time> Tag

<p>Published: <time datetime="2024-06-14">June 14, 2024</time></p>
<p>Event: <time datetime="2024-07-01T09:00">July 1, 2024 at 9:00 AM</time></p>

💡 datetime attribute machine-readable format mein hota hai — search engines aur calendars isse samajhte hain.
