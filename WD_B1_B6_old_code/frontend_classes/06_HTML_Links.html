1. <a> Tag — Anchor Tag (Link)

HTML mein links banane ke liye <a> (anchor) tag use karte hain. href attribute mein URL likhte hain.

📝 Basic Syntax:

<a href="URL">Link Text</a>

📝 Example:

<a href="https://www.google.com">Google Par Jayein</a>

🖥️ Output:

Google Par Jayein

2. Absolute vs Relative URLs

TypeExampleKab Use Karein
Absolute URL https://www.google.com/images External websites ke links ke liye
Relative URL about.html ya pages/contact.html Apni hi website ke pages ke links ke liye

📝 Code:

<!-- Absolute URL — doosri website -->
<a href="https://www.youtube.com">YouTube</a>

<!-- Relative URL — apni website ka page -->
<a href="contact.html">Contact Page</a>
<a href="pages/about.html">About Page (sub-folder mein)</a>
<a href="../index.html">Home (parent folder mein)</a>

3. Target Attribute — Link Kahan Khule

ValueBehavior
_selfSame tab mein khulega (default)
_blankNew tab mein khulega
_parentParent frame mein
_topFull window mein

📝 Code:

<!-- Same tab mein (default) -->
<a href="https://google.com">Same Tab</a>

<!-- New tab mein -->
<a href="https://google.com" target="_blank">New Tab Mein</a>

💡 Security Tip: target="_blank" ke saath rel="noopener noreferrer" bhi likhein — security ke liye.

4. Email aur Phone Links

📝 Code:

<!-- Email link — click par email app khulega -->
<a href="mailto:amir@codeacademy.com">Email Karein</a>

<!-- Email with subject -->
<a href="mailto:amir@codeacademy.com?subject=Admission Inquiry">
  Admission Ke Liye Email
</a>

<!-- Phone link — mobile par click se call lagegi -->
<a href="tel:+923001234567">Call Karein: 0300-1234567</a>

🖥️ Output:

📧 Email Karein

📞 Call Karein: 0300-1234567

5. Bookmark Links — Same Page Navigation

Ek hi page ke andar kisi specific section par jaane ke liye #id use karte hain:

📝 Code:

<!-- Navigation links -->
<a href="#section1">Section 1 Par Jayein</a>
<a href="#section2">Section 2 Par Jayein</a>

<!-- Target sections -->
<h2 id="section1">Section 1</h2>
<p>Yeh pehla section hai...</p>

<h2 id="section2">Section 2</h2>
<p>Yeh doosra section hai...</p>

<!-- Top par wapas jaane ka link -->
<a href="#">⬆ Top Par Jayein</a>

📌 Kaise kaam karta hai: Link mein #sectionId likhein, aur target element par id="sectionId" dein — click par page scroll kar jayega!

6. Image Ko Link Banana

<img> ko <a> ke andar rakhein — image clickable ban jayegi:

📝 Code:

<a href="https://google.com" target="_blank">
  <img src="https://via.placeholder.com/200x100" alt="Click Me">
</a>

7. Link Title — Hover Par Text

<a href="https://google.com" title="Google ki website par jayein">
  Google
</a>

🖥️ Output:

Google (hover karein tooltip ke liye)

8. Styled Navigation Menu Demo

Links ko CSS styling se navigation menu bana sakte hain:

🖥️ Live Demo:

🏠 Home 📖 Courses 👨‍🏫 Instructor 📞 Contact

📝 Code:

<style>
  .nav a {
    color: #00d2ff;
    text-decoration: none;
    padding: 8px 16px;
    border: 1px solid #00d2ff;
    border-radius: 5px;
    transition: 0.3s;
  }
  .nav a:hover {
    background: #00d2ff;
    color: #0f0f1a;
  }
</style>

<div class="nav">
  <a href="index.html">Home</a>
  <a href="courses.html">Courses</a>
  <a href="contact.html">Contact</a>
</div>

📝 Practice Tasks

  • Task 1: Ek navigation menu banayein jisme 5 links hon (Home, About, Services, Gallery, Contact). Har link hover par color change kare.
  • Task 2: Ek long page banayein jisme 5 sections hon. Top par bookmark links hon jo click par us section tak scroll karein. Bottom par "Back to Top" link ho.
  • Task 3: Email link aur phone link add karein apne contact section mein.
