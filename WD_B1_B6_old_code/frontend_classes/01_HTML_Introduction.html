1. HTML Kya Hai?

HTML ka full form hai HyperText Markup Language.

  • HyperText: Aisa text jis mein links hon — ek page se doosre page par ja sakein.
  • Markup: Tags ka use karke content ko structure dena — jaise heading, paragraph, image.
  • Language: Yeh ek language hai jo browser samajhta hai — lekin yeh programming language NAHI hai.

💡 Yaad rakhein: HTML ek markup language hai, programming language nahi. Yeh sirf content ko structure deti hai. Logic aur calculations ke liye JavaScript use hoti hai.

2. HTML Ki Mukhtasir History

Saal Version Kya Hua?
1991 HTML 1.0 Tim Berners-Lee ne pehli baar HTML banai — sirf 18 tags the
1995 HTML 2.0 Standard version release hua — forms add hue
1997 HTML 3.2 / 4.0 Tables, scripts, aur stylesheets ka support aaya
2014 HTML5 Modern version — audio, video, canvas, semantic tags sab add hue

Aaj kal hum HTML5 use karte hain jo sabse latest aur powerful version hai.

3. HTML Kahan Use Hota Hai?

HTML har jagah use hota hai jahan website ya web app banana ho:

  • 🌐 Websites: Google, Facebook, YouTube — sab HTML par based hain
  • 📱 Web Apps: Gmail, WhatsApp Web — HTML + CSS + JavaScript se bante hain
  • 📧 Emails: Rich HTML emails (images, buttons wale)
  • 📄 Documents: Online documents aur reports
  • 🎮 Games: Simple browser games bhi HTML5 Canvas se bante hain

4. Browser HTML Kaise Read Karta Hai?

Jab aap browser mein koi website kholte ho, yeh hota hai:

  1. Browser HTML file ko server se download karta hai
  2. HTML code ko parse (read) karta hai — upar se neeche
  3. Har tag ko samajh kar ek DOM Tree (Document Object Model) banata hai
  4. CSS styles apply karta hai
  5. Final result screen par render (display) karta hai

💡 Samjho: Browser ek "translator" hai — woh HTML code ko visual output mein convert karta hai.

5. Pehla HTML Program: Hello World 🎉

Chalein apna pehla HTML page likhte hain:

📝 Code:

<!DOCTYPE html>
<html>
<head>
  <title>Mera Pehla Page</title>
</head>
<body>
  <h1>Hello World!</h1>
  <p>Yeh mera pehla HTML page hai.</p>
</body>
</html>

🖥️ Browser Output:

Hello World!

Yeh mera pehla HTML page hai.

📖 Code Ka Explanation:

Code Kya Karta Hai
<!DOCTYPE html> Browser ko batata hai ke yeh HTML5 document hai
<html> Root element — sab kuch iske andar hota hai
<head> Metadata area — title, CSS, settings yahan hoti hain
<title> Browser ke tab mein dikhne wala title
<body> Visible content — jo screen par dikhta hai
<h1> Sabse bada heading
<p> Paragraph — normal text

6. HTML File Kaise Banate Hain?

  1. Apna code editor kholein (VS Code recommended hai)
  2. New file banayein
  3. File ko .html extension se save karein — jaise index.html
  4. HTML code likhein
  5. File ko save karein (Ctrl + S)
  6. File par double-click karein — browser mein khul jayegi!

⚠️ Zaroori: File ka extension .html hona chahiye — .txt ya kuch aur nahi!

8. HTML Tags Ka Basic Concept

HTML mein sab kuch tags se banta hai. Tags angle brackets < > mein likhay jate hain.

Tag Ki Structure:

<tagname> Content yahan aata hai </tagname>

Opening Tag:  <tagname>
Closing Tag:  </tagname>  (slash / ke saath)

Self-Closing Tags:

Kuch tags ko close nahi karna parta:

<br>      <!-- Line break -->
<hr>      <!-- Horizontal line -->
<img>     <!-- Image -->
<input>   <!-- Input field -->

Example:

<p>Yeh ek paragraph hai.</p>
<h1>Yeh heading hai.</h1>
<br>  <!-- Yeh self-closing hai -->
