1. <textarea> — Multi-Line Text Input

Jab bohat zyada text likhna ho (message, comments, bio) toh <textarea> use karein:

<label for="msg">Message:</label>
<textarea id="msg" rows="5" cols="40" placeholder="Apna message likhein..."></textarea>

🖥️ Demo:

📌 rows = visible lines ki tadaad. cols = visible width (characters). CSS se bhi control ho sakta hai.

2. <select> — Dropdown Menu

📝 Basic Dropdown:

<label for="city">City:</label>
<select id="city">
  <option value="" disabled selected>-- Select City --</option>
  <option value="nowshera">Nowshera</option>
  <option value="peshawar">Peshawar</option>
  <option value="islamabad">Islamabad</option>
  <option value="lahore">Lahore</option>
</select>

📝 Grouped Dropdown (optgroup):

<select>
  <optgroup label="Frontend">
    <option>HTML</option>
    <option>CSS</option>
    <option>JavaScript</option>
  </optgroup>
  <optgroup label="Backend">
    <option>Node.js</option>
    <option>Python</option>
  </optgroup>
</select>

3. Radio Buttons — Ek Option Select Karna

Jab sirf ek option select karwana ho. Sab radio buttons ka name same hona chahiye:

<p>Gender:</p>
<label><input type="radio" name="gender" value="male"> Male</label>
<label><input type="radio" name="gender" value="female"> Female</label>
<label><input type="radio" name="gender" value="other"> Other</label>

🖥️ Demo:

Gender:

📌 Key Rule: Same group ke radio buttons ka name attribute same hona chahiye — tabhi ek waqt mein sirf ek select hoga.

4. Checkboxes — Multiple Options Select Karna

Jab ek se zyada options select karwa sakte hain:

<p>Konse courses seekhne hain?</p>

<label><input type="checkbox" name="course" value="js"> JavaScript</label>
<label><input type="checkbox" name="course" value="react"> React</label>

7. <button> vs <input type="submit">

Feature<input type="submit"><button>
ContentSirf text (value attribute)Text + HTML (icons, images)
FlexibilityLimitedZyada flexible
StylingMushkilAasaan
 
<!-- Input submit — sirf text -->
<input type="submit" value="Submit">

<!-- Button — HTML content support karta hai -->
<button type="submit">
  🚀 Submit Form
</button>

💡 Recommendation: <button> use karein — zyada flexible aur powerful hai.

8. Form Validation Attributes

AttributePurposeExample
requiredField khali nahi chhod sakte<input required>
minlengthMinimum charactersminlength="5"
maxlengthMaximum charactersmaxlength="50"
min / maxNumber ka rangemin="18" max="65"
patternRegex pattern matchpattern="[0-9]{11}" (11 digits)

📝 Pattern Example (Pakistani Phone):

<input type="tel" pattern="03[0-9]{9}" 
       placeholder="03XXXXXXXXX" 
       title="Pakistani mobile number: 03 ke baad 9 digits">
