1. HTML Entities Kya Hain?

HTML mein kuch characters reserved hain (jaise < > &). Agar inhein directly likhein toh browser confuse ho jayega. In ke liye entities use karte hain.

Entity Format:

&entity_name;     <!-- Named entity -->
&#entity_number;  <!-- Numeric entity -->
&#xHEX;           <!-- Hexadecimal entity -->

2. Reserved Characters — Zaroori Entities

CharacterEntity NameEntity NumberResult

📝 Example — HTML Code Dikhana:

<p>HTML mein heading ke liye &lt;h1&gt; tag use karein.</p>

🖥️ Output:

HTML mein heading ke liye <h1> tag use karein.

<p>Temperature: 35&deg;C</p>
<p>Price: &#8377; 5000 &rarr; Discounted: &#8377; 3500</p>

<p>Rating: &#11088;&#11088;&#11088;&#11088;&#11088;</p>
