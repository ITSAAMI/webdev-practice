1. <audio> Tag — Audio Play Karna

<!-- Basic audio player with controls -->
<audio controls>
  <source src="audio/song.mp3" type="audio/mpeg">
  <source src="audio/song.ogg" type="audio/ogg">
  Aapka browser audio support nahi karta.
</audio>

Audio Attributes:

AttributeKya Karta Hai
controlsPlay, pause, volume buttons dikhata hai
autoplayPage load par automatic play (muted ke saath)
loopAudio khatam hone par dobara start
mutedBy default mute
preloadauto / metadata / none

Audio Formats:

FormatMIME TypeSupport
MP3audio/mpeg✅ Sab browsers
OGGaudio/oggFirefox, Chrome, Opera
WAVaudio/wav✅ Sab browsers (large files)

💡 Multiple <source> tags dein — browser jo format support karega woh play karega. MP3 sabse safe choice hai.

2. <video> Tag — Video Play Karna

<video width="640" height="360" controls poster="thumbnail.jpg">
  <source src="video/intro.mp4" type="video/mp4">
  <source src="video/intro.webm" type="video/webm">
  Aapka browser video support nahi karta.
</video>

Video Attributes:

AttributeKya Karta Hai
controlsPlay/pause, volume, fullscreen buttons
autoplayAuto play (muted zaroori hai Chrome mein)
loopVideo repeat hota rahega
mutedBy default mute
posterVideo play hone se pehle thumbnail image
width / heightVideo ki size
preloadPage load par video kitna download ho

Video Formats:

FormatMIME TypeBest For
MP4 (H.264)video/mp4✅ Universal — sabse safe
WebMvideo/webmChrome, Firefox — chhota size
OGGvideo/oggFirefox, Chrome

Background Video (Muted, Autoplay, Loop):

<video autoplay muted loop style="width:100%;">
  <source src="bg-video.mp4" type="video/mp4">
</video>

⚠️ Chrome Rule: Autoplay sirf muted ke saath kaam karta hai — bina mute ke browser block kar dega.

3. <track> Tag — Subtitles / Captions

<video controls width="640">
  <source src="video.mp4" type="video/mp4">
  
  <track src="subtitles-en.vtt" kind="subtitles" srclang="en" label="English">
  <track src="subtitles-ur.vtt" kind="subtitles" srclang="ur" label="Urdu" default>
</video>

VTT File Format (subtitles-ur.vtt):

WEBVTT

00:00:01.000 --> 00:00:05.000
Assalam-o-Alaikum! Code Academy mein khush aamdeed.

00:00:06.000 --> 00:00:10.000
Aaj hum HTML5 seekhenge.
kind ValuePurpose
subtitlesTranslation subtitles
captionsSame language captions (deaf users ke liye)
descriptionsVisual content ki description (blind users ke liye)
chaptersVideo ke chapters/sections

4. Responsive Video

<!-- Method 1: Simple CSS -->
<video controls style="max-width: 100%; height: auto;">
  <source src="video.mp4" type="video/mp4">
</video>

<!-- Method 2: 16:9 Aspect Ratio Container -->
<div style="position:relative; padding-bottom:56.25%; height:0; overflow:hidden;">
  <video controls style="position:absolute; top:0; left:0; width:100%; height:100%;">
    <source src="video.mp4" type="video/mp4">
  </video>
</div>

5. JavaScript Se Audio/Video Control

<video id="myVideo" src="video.mp4"></video>

<button onclick="document.getElementById('myVideo').play()">▶ Play</button>
<button onclick="document.getElementById('myVideo').pause()">⏸ Pause</button>
<button onclick="document.getElementById('myVideo').volume = 0.5">🔉 Half Volume</button>
<button onclick="document.getElementById('myVideo').currentTime = 0">⏮ Restart</button>

Useful Properties aur Methods:

Property/MethodKya Karta Hai
.play()Video/audio play karta hai
.pause()Pause karta hai
.volumeVolume set (0.0 to 1.0)
.currentTimeCurrent position (seconds mein)
.durationTotal length (seconds)
.mutedtrue/false — mute toggle
.playbackRateSpeed (1.0 = normal, 2.0 = 2x)

  • Task 2: Ek background video hero section banayein — video autoplay, muted, loop ho. Upar text overlay ho.
  • Task 3: Custom controls banayein JavaScript se — Play, Pause, Mute, Volume Up, Volume Down buttons.
