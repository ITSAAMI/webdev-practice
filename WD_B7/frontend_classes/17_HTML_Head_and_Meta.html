🧠 Topic 17: Head & Meta Tags Intermediate

Code Academy Nowshera | Instructor: Syed Amir Hayat | Batch: FS_B10

1. <head> Section Ka Purpose

Head section mein metadata hota hai — jo screen par nahi dikhta lekin browser, search engines, aur social media ke liye zaroori hai.

Complete Head Template:

<head>
  <!-- Character Set -->
  <meta charset="UTF-8">
  
  <!-- Responsive Design -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <!-- SEO -->
  <title>Code Academy Nowshera - Web Development Courses</title>
  <meta name="description" content="Learn HTML, CSS, JavaScript at Code Academy Nowshera">
  <meta name="author" content="Syed Amir Hayat">
  
  <!-- Favicon -->
  <link rel="icon" href="favicon.ico" type="image/x-icon">
  
  <!-- CSS -->
  <link rel="stylesheet" href="css/style.css">
  
  <!-- Open Graph (Social Media) -->
  <meta property="og:title" content="Code Academy Nowshera">
  <meta property="og:description" content="Best coding academy in Nowshera">
  <meta property="og:image" content="https://example.com/og-image.jpg">
</head>

2. <meta charset="UTF-8">

UTF-8 character encoding se tamam languages ke characters support hote hain — English, Urdu, Arabic, Chinese sab.

<meta charset="UTF-8">

⚠️ Yeh hamesha sabse pehle head mein likhein — warna special characters toot sakte hain.

3. Viewport Meta Tag — Responsive Design

<meta name="viewport" content="width=device-width, initial-scale=1.0">
ValueMatlab
width=device-widthPage ki width device ki screen jitni ho
initial-scale=1.0Page 100% zoom par khule (zoomed out nahi)

❌ Bina viewport: Mobile par page chhota sa dikhta hai (desktop view).
✅ Viewport ke saath: Page mobile screen ke mutabiq adjust hota hai.

4. <title> Tag — SEO Ka King

<title>Code Academy Nowshera - Learn Web Development</title>

Title 3 jagah dikhta hai:

  • 🔖 Browser tab mein
  • 🔍 Google search results mein blue link ke taur par
  • ⭐ Bookmarks mein save hone wala naam

Best Practices:

  • 50-60 characters rakhein
  • Primary keyword pehle likhein
  • Har page ka title unique ho
  • Format: Topic - Site Name

5. Meta Description — Google Snippet

<meta name="description" 
      content="Code Academy Nowshera offers professional web development courses 
               including HTML, CSS, JavaScript, React, and Node.js. Join today!">

Yeh Google search results mein title ke neeche chhota sa text dikhata hai.

Best Practices:

  • 150-160 characters rakhein
  • Compelling aur descriptive likhein
  • Call-to-action shamil karein ("Join today!", "Learn more")
  • Har page ka description unique ho

6. Open Graph Tags — Social Media Preview

Jab aap link WhatsApp, Facebook, Twitter par share karein toh acha preview dikhane ke liye:

<meta property="og:title" content="Code Academy Nowshera">
<meta property="og:description" content="Best coding academy in KPK">
<meta property="og:image" content="https://example.com/preview.jpg">
<meta property="og:url" content="https://codeacademy-nowshera.com">
<meta property="og:type" content="website">

<!-- Twitter specific -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Code Academy Nowshera">

💡 OG image ka recommended size: 1200 x 630 pixels. Isse acha preview aayega WhatsApp aur Facebook par.

7. Favicon — Browser Tab Icon

<!-- .ico format (classic) -->
<link rel="icon" href="favicon.ico" type="image/x-icon">

<!-- .png format (modern) -->
<link rel="icon" href="images/favicon.png" type="image/png">

<!-- Different sizes -->
<link rel="icon" sizes="32x32" href="favicon-32x32.png">
<link rel="icon" sizes="16x16" href="favicon-16x16.png">

<!-- Apple Touch Icon (iPhone bookmark ke liye) -->
<link rel="apple-touch-icon" href="apple-touch-icon.png">

8. Other Head Elements

<!-- External CSS -->
<link rel="stylesheet" href="css/style.css">

<!-- Google Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Poppins&display=swap" rel="stylesheet">

<!-- Internal CSS -->
<style>
  body { font-family: 'Poppins', sans-serif; }
</style>

<!-- JavaScript -->
<script src="js/script.js" defer></script>

<!-- Base URL -->
<base href="https://www.codeacademy.com/">

<!-- Auto Refresh (5 seconds baad) -->
<meta http-equiv="refresh" content="5">

<!-- Redirect (3 seconds baad doosre page par) -->
<meta http-equiv="refresh" content="3;url=https://newsite.com">

💡 defer attribute JS file ko page load hone ke baad run karta hai — page fast load hota hai.

📝 Practice Tasks

  • Task 1: Ek page ka complete head section likhein — charset, viewport, title, description, author, favicon, aur Open Graph tags sab shamil hon.
  • Task 2: 3 alag pages banayein (home, about, contact) — har ek ka title aur description unique ho.
  • Task 3: Google Fonts se ek font import karein head mein aur apne page par apply karein.
