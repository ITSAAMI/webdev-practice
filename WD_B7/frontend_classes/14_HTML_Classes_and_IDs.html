1. Class Attribute — Reusable Styling

class se ek same style multiple elements par lagta hai:

<style>
  .highlight-text { color: #e94560; font-weight: bold; background: #ffe0e6; padding: 5px 10px; border-radius: 4px; }
</style>

<p class="highlight-text">Yeh pehla highlighted paragraph hai.</p>
<p class="highlight-text">Yeh doosra bhi same class se styled hai.</p>
<p>Yeh normal paragraph hai — koi class nahi.</p>

🖥️ Output:

Yeh pehla highlighted paragraph hai.

Yeh doosra bhi same class se styled hai.

Yeh normal paragraph hai — koi class nahi.

💡 CSS mein class select karne ke liye dot (.) use karein: .highlight-text { }

2. ID Attribute — Unique Identifier

id attribute unique hota hai — ek page mein ek id sirf ek baar use ho sakti hai:

<style>
  #main-title { color: #0f3460; font-size: 2rem; text-align: center; border-bottom: 3px solid #e94560; padding-bottom: 10px; }
</style>

<h1 id="main-title">Code Academy Nowshera</h1>

🖥️ Output:

Code Academy Nowshera

💡 CSS mein id select karne ke liye hash (#) use karein: #main-title { }

4. Multiple Classes Ek Element Par

Ek element ko multiple classes de sakte hain — space se separate karein:

<style>
  .card-box { background: white; padding: 20px; border-radius: 8px; margin: 10px 0; }
  .shadow { box-shadow: 0 4px 15px rgba(0,0,0,0.2); }
  .border-red { border-left: 4px solid #e94560; }
  .border-blue { border-left: 4px solid #0f3460; }
</style>

<!-- Dono cards ko card-box + shadow milega, lekin border color alag -->
<div class="card-box shadow border-red">Red Border Card</div>
<div class="card-box shadow border-blue">Blue Border Card</div>

🖥️ Output:

Red Border Card
Blue Border Card

5. Naming Conventions — Best Practices

ConventionExamplePopular In
kebab-casemain-header, nav-linkCSS (most recommended)
camelCasemainHeader, navLinkJavaScript
snake_casemain_header, nav_linkPython, databases
BEMblock__element--modifierLarge projects

❌ Galat Naming:

class="1header"       <!-- Number se start nahi ho sakta -->
class="my header"     <!-- Space NAHI aa sakta -->
class="header!@#"     <!-- Special characters avoid karein -->

✅ Sahi Naming:

class="main-header"   <!-- kebab-case ✅ -->
class="card-title"    <!-- descriptive ✅ -->
id="contact-form"     <!-- meaningful ✅ -->

6. JavaScript Mein ID aur Class Ka Use

<!-- ID se ek element select karna -->
<script>
  let title = document.getElementById("main-title");
  title.style.color = "red";
</script>

<!-- Class se multiple elements select karna -->
<script>
  let cards = document.getElementsByClassName("card-box");
  // cards[0], cards[1], etc.
</script>

<!-- Modern way: querySelector -->
<script>
  document.querySelector("#main-title");      // ID
  document.querySelectorAll(".card-box");     // All classes
</script>
