📄 Topic 04: Paragraphs & Line Breaks

1. <p> Tag — Paragraphs

Paragraph tag <p> se hum text ko alag alag blocks mein likhte hain. Har paragraph ke baad browser automatic space deta hai.

📝 Code:

<p>Yeh pehla paragraph hai. Code Academy Nowshera mein HTML seekhein.</p>
<p>Yeh doosra paragraph hai. Dono ke beech mein gap aayega.</p>

🖥️ Output:

Yeh pehla paragraph hai. Code Academy Nowshera mein HTML seekhein.

Yeh doosra paragraph hai. Dono ke beech mein gap aayega.

2. Browser Whitespace Kaise Handle Karta Hai

HTML mein chahe aap kitne bhi spaces ya enter dein, browser sirf ek space dikhata hai:

📝 Code:

<p>Yeh     bohat      spaces     hain      lekin
browser sirf      ek          space          dikhayega.</p>

🖥️ Output:

Yeh bohat spaces hain lekin browser sirf ek space dikhayega.

💡 Browser multiple spaces ko ek space mein convert karta hai aur enter/newline ko bhi ignore karta hai.

3. <br> Tag — Line Break

Agar aapko text ko next line par le jaana hai bina naya paragraph banaye, toh <br> use karein:

📝 Code:

<p>
  Mera naam Amir hai.<br>
  Main Nowshera se hoon.<br>
  Main ek instructor hoon.
</p>

🖥️ Output:

Mera naam Amir hai.
Main Nowshera se hoon.
Main ek instructor hoon.

📌 <br> ek self-closing tag hai — isay close nahi karte. <br/> bhi chalega.

4. <hr> Tag — Horizontal Line

<hr> se page par horizontal line aati hai — sections separate karne ke liye:

📝 Code:

<h2>Section 1</h2>
<p>Yeh pehla section hai.</p>
<hr>
<h2>Section 2</h2>
<p>Yeh doosra section hai.</p>

🖥️ Output:

Section 1

Yeh pehla section hai.

Section 2

Yeh doosra section hai.

HR Ko Style Karna:

<hr style="border: 2px solid red; width: 50%;">
<hr style="border: none; border-top: 3px dashed blue;">

5. <pre> Tag — Preformatted Text

<pre> tag text ko jaise likhein waise dikhata hai — spaces, tabs, aur newlines sab preserve hoti hain:

📝 Code:

<pre>
  Naam:     Syed Amir
  City:     Nowshera
  Role:     Instructor
  
  Line 1
  Line 2
  Line 3
</pre>

🖥️ Output:

  Naam:     Syed Amir
  City:     Nowshera
  Role:     Instructor
  
  Line 1
  Line 2
  Line 3

💡 <pre> tag monospace font use karta hai aur spaces exactly waise dikhata hai jaise aap ne type kiye hain.

6. &nbsp; — Non-Breaking Space

Agar aapko multiple spaces chahiyein, toh &nbsp; entity use karein:

📝 Code:

<p>Normal:   spaces    ignored    hon    ge</p>
<p>Fixed:&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5 extra spaces</p>

🖥️ Output:

Normal: spaces ignored hon ge

Fixed:     5 extra spaces
