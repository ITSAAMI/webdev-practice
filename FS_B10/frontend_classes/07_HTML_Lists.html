1. Unordered Lists (<ul>)

Shomaila Hamraz, jab humne aisi list banani ho jis mein tarteeb (numbering) ki zaroorat na ho tou hum <ul> (Unordered List) ka use karte hain. Har item likhne ke liye uske andar <li> (List Item) lagaya jata hai. By default isme bullet points atay hain. 
<h3>Frontend Skills:</h3>
<ul>
    <li>HTML5</li>
    <li>CSS3</li>
    <li>JavaScript</li>
</ul>
Live Browser Output

Frontend Skills:

  • HTML5
  • CSS3
  • JavaScript

2. Ordered Lists (<ol>)

M.Zuhaib dhayan dein: Agar aap recipe likh rahe hain ya koi step-by-step process bta rahe hain jahan numbering zaroori hai, tou wahan <ol> (Ordered List) istemal hoti hai. Isme automatically 1, 2, 3.. numbers atay hain. 
<h3>Chai Banane Ka Tariqa:</h3>
<ol>
    <li>Pani ubaalen</li>
    <li>Patti aur cheeni dalen</li>
    <li>Doodh shamil karen</li>
</ol>
Live Browser Output

Chai Banane Ka Tariqa:

  1. Pani ubaalen
  2. Patti aur cheeni dalen
  3. Doodh shamil karen

3. Description Lists (<dl>)

Ek aur qisam ki list hoti hai jise Description List kaha jata hai. Isme numbers ya bullets nahi hote, balke terms aur unki definitions hoti hain (jaise dictionary mein).
<dl> = Description List
<dt> = Data Term (Naam)
<dd> = Data Definition (Wazahat) 
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>
Live Browser Output
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

4. Nested Lists (List ke andar List)

Syed Muhammad Subtain Gul, aap ek list ke list item <li> ke andar ek aur list (ul ya ol) bana sakte hain. Isay Nested list kehte hain. Yeh menus aur sub-menus banane mein bohat kaam aati hai. 
<ul>
    <li>Fruits
        <!-- Nayi list pichle <li> ke close hone se pehle aye gi -->
        <ul>
            <li>Apple</li>
            <li>Banana</li>
        </ul>
    </li>
    <li>Vegetables</li>
</ul>
Live Browser Output
  • Fruits
    • Apple
    • Banana
  • Vegetables
