1. HTML Links (<a> Tag)

Abdul Muheman Shah, web pages ko ek doosre ke sath jorne (connect karne) ke liye hum Links ka istemal karte hain. HTML mein link banane ke liye <a> (Anchor) tag use hota hai.

Iska sab se ahem hissa href (Hypertext Reference) attribute hai, jisme hum batate hain ke click karne par kahan jana hai. 
<a href="https://github.com/ITSAAMI">Code Academy GitHub</a>

2. Absolute vs Relative URLs

Sudais Ahmad dhyan de: Links do tarah ke hote hain:
1. Absolute URL: Poora address likhna parhta hai (e.g., https://www.google.com). Yeh doosri websites par jane ke liye hota hai.
2. Relative URL: Isme "https" ya website ka naam nahi likhte. Sirf apne he folder ki doosri file ka naam likhte hain (e.g., about.html). Yeh apni he website ke pages ko link karne ke liye hota hai. 
<!-- Absolute URL (Doosri website) -->
<a href="https://www.google.com">Google par jayen</a>

<!-- Relative URL (Apni website ka dosra page) -->
<a href="04_HTML_Text_Formatting.html">Pichla Lesson dekhein</a>

3. Link ko New Tab mein open karna

Mobina Farooq Qureshi, by default link click karne se wohi page open ho jata hai aur purana page gayab. Agar aap chahti hain ke link click hone par ek naya tab (new window) open ho, tou target="_blank" attribute ka istemal karein. 
<a href="https://logicgigs.com" target="_blank">LogicGigs (Open in new tab)</a>

4. Email aur Phone Links

Aqsa, website par aise links bhi banaye ja sakte hain jin par click karne se seedha user ki Email app ya Phone dialer khul jaye. Iske liye hum mailto: aur tel: ka istemal karte hain. 
<!-- Email Link -->
<a href="mailto:amir@codeacademy.com">Mujhe Email karein</a>

<!-- Phone Number Link -->
<a href="tel:+923001234567">Mujhe Call karein</a>
