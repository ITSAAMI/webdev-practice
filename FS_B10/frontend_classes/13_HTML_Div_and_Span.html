1. <div> — Block Container

<div> ek generic block container hai. Iska apna koi visual effect nahi — sirf elements ko group karta hai:

<div style="background: #f0f0f0; padding: 20px; border-radius: 8px;">
  <h2>About Section</h2>
  <p>Yeh about section hai.</p>
</div>

2. <span> — Inline Container

<span> ek inline container hai — text ke chhote hisse ko alag style dene ke liye:

<p>
  Mera naam <span style="color: red; font-weight: bold;">Syed Amir</span> hai
  aur main <span style="color: blue;">Nowshera</span> se hoon.
</p>

3. <div> vs <span> — Comparison

Feature<div><span>
Display TypeBlockInline
WidthFull width (100%)Content jitna
New LineHaanNahi
Use CaseSections, layouts, containersText ke hisson ko style karna
Semantic MeaningKoi nahi — genericKoi nahi — generic

4. Div Se Page Layout

📝 Basic Page Layout:

<div id="wrapper" style="max-width: 900px; margin: 0 auto;">

  <div id="header" style="background: #0f3460; color: white; padding: 20px;">
    <h1>My Website</h1>
  </div>

  <div id="nav" style="background: #16213e; padding: 10px;">
    Home | About | Contact
  </div>

  <div id="content" style="display: flex;">
    <div id="main" style="flex: 3; padding: 20px;">
      <h2>Main Content</h2>
      <p>Article content yahan...</p>
    </div>
    <div id="sidebar" style="flex: 1; background: #f0f0f0; padding: 20px;">
      <h3>Sidebar</h3>
      <p>Links, ads...</p>
    </div>
  </div>

  <div id="footer" style="background: #0f3460; color: white; padding: 10px; text-align: center;">
    &copy; 2024 Code Academy
  </div>

</div>

5. Div vs Semantic Elements

HTML5 mein semantic tags aaye hain jo div se better hain:

Div ApproachSemantic Approach
<div id="header"><header>
<div id="nav"><nav>
<div id="main"><main>
<div id="sidebar"><aside>
<div id="footer"><footer>
<div class="article"><article>

💡 Best Practice: Jab bhi possible ho, semantic tags use karein. <div> sirf tab use karein jab koi semantic tag fit na ho.

📝 Practice Tasks

  • Task 1: Sirf <div> tags use karke ek complete page layout banayein — Header, Navigation, Main Content, Sidebar, Footer. CSS se styled karein.
  • Task 2: Ek paragraph mein <span> use karke 3 alag words ko alag colors mein style karein.
  • Task 3: Nested divs se 3 cards banayein — har card mein title, image placeholder, aur description ho.
