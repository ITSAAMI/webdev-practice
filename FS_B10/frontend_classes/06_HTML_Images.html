1. Images lagana (<img> Tag)

Moizza aur Sanabil, website par tasweer lagane ke liye hum <img> tag ka use karte hain. Yeh tag empty hota hai yani iska koi closing tag (</img>) nahi hota.

Isme src (source) attribute sab se zaroori hai, jo batata hai ke image ki file kahan par pari hai. 
<!-- Example 1: Same folder se image uthana -->
<img src="profile-pic.jpg">

<!-- Example 2: Internet se image uthana (Absolute URL) -->
<img src="https://via.placeholder.com/300x150/2ecc71/ffffff?text=HTML+Images">

2. Alt Attribute (Bohot Zaroori!)

Sameer Amir, hamesha apne <img> tag ke andar alt (Alternate text) attribute zarur lagayen.
Iske 2 faide hain:
1. Agar internet slow ho aur image load na ho sake, tou yeh text nazar aayega.
2. Blind log (jo screen readers use karte hain), unko image ke bare mein pata chalega. 
<img src="wrong-image-path.jpg" alt="Sameer Amir ki Profile Picture">

3. Image ka Size (Width aur Height)

Aap width aur height attributes de kar image ka size pixels mein control kar sakte hain. Lekin modern design mein hum size control karne ke liye CSS ko tarjeeh (preference) dete hain. 
<img src="logo.png" alt="Academy Logo" width="200" height="100">

4. Image ko Link bananan

Junaid Ahmad, agar aap chahte hain ke user kisi image par click kare tou woh dosre page par chala jaye, tou aapko sirf <img> tag ko <a> (link) tag ke andar rakhna hoga. 
<a href="https://github.com/ITSAAMI" target="_blank">
    <img src="https://via.placeholder.com/150/2c3e50/ffffff?text=Click+Me!" alt="Clickable Image">
</a>
