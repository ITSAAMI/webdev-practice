1. Web Storage API Kya Hai?

HTML5 Web Storage se aap user ke browser mein locally data save kar sakte hain (bina database/server ke). Yeh cookies se zyada secure aur fast hai.

Storage TypeData Kab Tak Rehta Hai?Capacity
localStoragePermanent (browser band karne ke baad bhi rehta hai)~5MB+
sessionStorageTemporary (tab/window close karte hi delete ho jata hai)~5MB+
Cookies (Old)Expiry date takSirf 4KB

2. Local Storage — Basic Methods

<script>
  // 1. Data Save Karna (set)
  localStorage.setItem("username", "Amir Hayat");

  // 2. Data Read Karna (get)
  let user = localStorage.getItem("username");
  console.log(user); // Output: Amir Hayat

  // 3. Ek item Delete Karna
  localStorage.removeItem("username");

  // 4. Poora Storage Clear Karna (Sab delete)
  localStorage.clear();
</script>

⚠️ Note: Web Storage hamesha data ko String (text) format mein save karta hai. Agar numbers ya arrays/objects save karne hon toh JSON use karna padta hai.

function updateDisplay() {
  let savedName = localStorage.getItem("demoName");
  document.getElementById("displayName").innerText = savedName;
}

4. Session Storage

sessionStorage ke methods bilkul same hain, bas data life alag hai:

<script>
  // Tab band hote hi yeh delete ho jayega
  sessionStorage.setItem("themeColor", "Dark");
  
  let theme = sessionStorage.getItem("themeColor");
  console.log(theme);
</script>
