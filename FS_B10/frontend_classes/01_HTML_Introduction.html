1. HTML Kya Hai?

HTML (HyperText Markup Language) web pages banane ke liye istemal hoti hai. Yeh kisi bhi website ka "skeleton" (dhancha) hoti hai. Browser (jaise Chrome) HTML code ko parhta hai aur screen par text, images, aur links dikhata hai.

HTML mein Tags hote hain jo browser ko batatay hain ke content kaisa dikhna chahiye (e.g., heading, paragraph, button).

2. Basic HTML Structure

Har HTML page ka ek khaas structure hota hai jisme <head> (background information ke liye) aur <body> (screen par dikhne wale content ke liye) shamil hote hain. Chaliye dekhte hain ke Sameer Amir ka pehla web page kaisa dikhta hai: 
<!DOCTYPE html>
<html>
<head>
    <title>Sameer Amir's First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Mera naam Sameer Amir hai aur main HTML seekh raha hoon.</p>
</body>
</html>
Live Browser Output

Hello World!

Mera naam Sameer Amir hai aur main HTML seekh raha hoon.

3. Important Tags ki Pehchan

Moizza aur Sanabil, yaad rakhiye ke HTML tags hamesha angle brackets < > mein likhe jate hain. Zyadatar tags joray (pairs) mein aate hain: ek start tag aur ek end tag jisme slash / hota hai.
  • <!DOCTYPE html> — Browser ko batata hai ke yeh HTML5 version hai.
  • <html> ... </html> — Yeh root element hai, baqi sab tags iske andar aate hain.
  • <head> ... </head> — Isme website ka Title aur baqi settings hoti hain jo page pe samne nahi dikhti.
  • <body> ... </body> — Iske andar jo kuch bhi likha jata hai, woh user ko screen par nazar aata hai.
