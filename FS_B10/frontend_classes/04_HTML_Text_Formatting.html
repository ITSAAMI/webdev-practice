Topic 04: HTML Text Formatting

Code Academy Nowshera | Instructor: Syed Amir Hayat Kakakhel

Level: Beginner | Batch: FS_B10

1. Bold & Strong Text (<b> and <strong>)

Sabeel Mehmood, kisi text ko mota (bold) karne ke liye hum <b> ya <strong> tag ka istemal karte hain. Dikhne mein dono ek jaise lagte hain, lekin <strong> screen readers ko batata hai ke yeh text bohat aham (important) hai. 
<p>Yeh ek <b>bold</b> text hai.</p>
<p>Sabeel ki file mein yeh <strong>important</strong> baat hai.</p>
Live Browser Output

Yeh ek bold text hai.

Sabeel ki file mein yeh important baat hai.

2. Italic & Emphasized Text (<i> and <em>)

Text ko thora tircha (italic) karne ke liye <i> tag use hota hai. Lekin agar kisi lafz par khaas zor (emphasis) dena ho (jaise bolne mein zor diya jata hai), tou <em> tag behtar rehta hai. 
<p>Yeh text <i>italic</i> style mein hai.</p>
<p>Shahzaib Sajjad ne kaha tha: <em>zaroor aana!</em></p>
Live Browser Output

Yeh text italic style mein hai.

Shahzaib Sajjad ne kaha tha: zaroor aana!

3. Underline, Inserted aur Highlighted Text

Abdur Rehman dhayan se dekhna: Kisi lafz ke neechay line lagani ho tou <u> ya <ins> tag use hota hai. Kisi lafz ko highlighter jaisa dikhana ho (background color yellow) tou <mark> tag use karte hain. 
<p>Is hisse ko <u>underline</u> karo.</p>
<p>Naya added text: <ins>Free Course</ins></p>
<p>Mujhe <mark>HTML seekhni hai</mark>.</p>
Live Browser Output

Is hisse ko underline karo.

Naya added text: Free Course

Mujhe HTML seekhni hai.

4. Math/Science aur Deleted Text

Junaid Ahmad, agar aapne Math ka formula (jaise a2) ya science (H2O) likhna ho tou <sup> (superscript, upar) aur <sub> (subscript, neechay) tags kaam aate hain. Kisi text ko kata hua dikhana ho tou <del> use hota hai. 
<p>Pani ka formula H<sub>2</sub>O hai.</p>
<p>Algebra formula: x<sup>2</sup> + y<sup>2</sup> = z<sup>2</sup></p>
<p>Course fee: <del>Rs. 10000</del> <ins>Free!</ins></p>
Live Browser Output

Pani ka formula H2O hai.

Algebra formula: x2 + y2 = z2

Course fee: Rs. 10000 Free!
