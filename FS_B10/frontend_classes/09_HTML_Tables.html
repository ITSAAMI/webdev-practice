1. Basic Table Structure

Table ke 4 basic tags hain:

TagKya Karta Hai
<table>Table container
<tr>Table Row — ek horizontal line
<th>Table Header — bold + centered
<td>Table Data — normal cell

📝 Code:

<table border="1">
  <tr>
    <th>Naam</th>
    <th>City</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ahmad</td>
    <td>Nowshera</td>
    <td>20</td>
  </tr>
  <tr>
    <td>Sara</td>
    <td>Peshawar</td>
    <td>22</td>
  </tr>
</table>

🖥️ Output:

NaamCityAge
AhmadNowshera20
SaraPeshawar22

2. Table Sections — thead, tbody, tfoot

<table>
  <caption>Student Results — Batch FS_B10</caption>
  <thead>
    <tr><th>Naam</th><th>HTML</th><th>CSS</th><th>Total</th></tr>
  </thead>
  <tbody>
    <tr><td>Ali</td><td>85</td><td>90</td><td>175</td></tr>
    <tr><td>Fatima</td><td>92</td><td>88</td><td>180</td></tr>
  </tbody>
  <tfoot>
    <tr><td><strong>Average</strong></td><td>88.5</td><td>89</td><td>177.5</td></tr>
  </tfoot>
</table>

🖥️ Output:

Student Results — Batch FS_B10
NaamHTMLCSSTotal
Ali8590175
Fatima9288180
Average88.589177.5

3. Colspan aur Rowspan — Cells Merge Karna

AttributeKya Karta Hai
colspan="2"2 columns ko merge (horizontally)
rowspan="3"3 rows ko merge (vertically)

📝 Colspan Example:

<table border="1">
  <tr>
    <th colspan="3">Student Information</th>
  </tr>
  <tr>
    <th>Naam</th><th>Class</th><th>Marks</th>
  </tr>
  <tr>
    <td>Usman</td><td>10th</td><td>450</td>
  </tr>
</table>

🖥️ Output:

Student Information
NaamClassMarks
Usman10th450

📝 Rowspan Example:

<table border="1">
  <tr>
    <th>Department</th><th>Naam</th><th>Marks</th>
  </tr>
  <tr>
    <td rowspan="2">Science</td>
    <td>Ali</td><td>90</td>
  </tr>
  <tr>
    <td>Ayesha</td><td>85</td>
  </tr>
  <tr>
    <td rowspan="2">Arts</td>
    <td>Hassan</td><td>78</td>
  </tr>
  <tr>
    <td>Zainab</td><td>82</td>
  </tr>
</table>

🖥️ Output:

DepartmentNaamMarks
ScienceAli90
Ayesha85
ArtsHassan78
Zainab82

4. CSS Se Table Styling — Zebra Stripes

<style>
  table { width: 100%; border-collapse: collapse; }
  th { background: #0f3460; color: white; padding: 12px; }
  td { padding: 10px; border: 1px solid #ddd; }
  tr:nth-child(even) { background: #f2f2f2; }
  tr:hover { background: #ddd; }
</style>

💡 border-collapse: collapse; borders ko merge karta hai — bina iske double borders dikhte hain.

5. Responsive Table

Mobile par table ko horizontally scrollable banayein:

<div style="overflow-x: auto;">
  <table>
    <!-- Table content -->
  </table>
</div>

📌 Table ko <div> mein wrap karein aur overflow-x: auto dein — chhoti screen par scroll ho jayega.

📝 Practice Tasks

  • Task 1: Apni class ke 5 students ka result table banayein — columns: Name, HTML Marks, CSS Marks, JS Marks, Total, Grade. thead, tbody, tfoot use karein.
  • Task 2: Timetable banayein (Monday-Friday, 8am-2pm) — colspan aur rowspan dono use karein.
  • Task 3: Table ko CSS se beautiful banayein — zebra stripes, hover effects, aur responsive wrapper.
