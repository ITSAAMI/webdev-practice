1. Tag aur Element mein kya farq hai?

Syed Muhammad Subtain Gul aur baqi sab students dhayan dein: Aksar log "Tag" aur "Element" ko ek he cheez samajhte hain, magar in mein thora farq hai.

Tag: Sirf brackets wala hissa (e.g., <p>).
Element: Start tag, uske andar ka content, aur end tag... in teeno cheezon ko mila kar ek "Element" banta hai.
Start Tag Element Content End Tag
<h1> My First Heading </h1>
<p> My first paragraph. </p> 
<!-- Yeh poora mila kar ek HTML Element kehlata hai -->
<h1>Insha Mashal's Website</h1>

2. Nested HTML Elements

Rukhsana Ismail, "Nesting" ka matlab hota hai ek cheez ke andar doosri cheez dalna. HTML elements ko ek doosre ke andar rakha ja sakta hai. Darasal, har HTML page nested elements se mil kar he banta hai. 
<html>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

Yahan <h1> aur <p> element <body> ke andar nested hain, aur <body> <html> ke andar nested hai.

3. Empty HTML Elements (Self-Closing)

Kuch HTML elements aisay hote hain jin ke andar koi content (text) nahi hota aur na he unka koi End tag (jaise </br>) hota hai. Inhein hum Empty Elements ya Self-closing tags kehte hain. 
<!-- <br> tag line break (nayee line) ke liye use hota hai -->
<p>Yeh pehli line hai.<br>Yeh doosri line hai.</p>

<!-- <hr> tag ek horizontal line draw karne ke liye use hota hai -->
<hr>
Live Browser Output

Yeh pehli line hai.
Yeh doosri line hai.

4. Kya HTML Case Sensitive Hai?

Irwa Ali, HTML tags "case sensitive" nahi hote. Iska matlab yeh hai ke <P> aur <p> dono same hain. Lekin achay developers hamesha lowercase (chhoti ABCD) istemal karte hain. Code Academy Nowshera mein hum bhi lowercase he follow karenge. 
<!-- ❌ Aisa mat likhein (Kaam karega but best practice nahi hai) -->
<P>Mera Paragraph</P>

<!-- ✅ Hamesha lowercase mein likhein (Professional way) -->
<p>Mera Paragraph</p>
