# 02 - To-Do List

A fully functional to-do list application with task management, filtering, and persistent storage.

## Features

✅ Add new tasks
✅ Mark tasks as complete/incomplete
✅ Delete tasks
✅ Filter tasks (All, Active, Completed)
✅ Clear all completed tasks
✅ Task counter
✅ Persistent storage (localStorage)
✅ Responsive design
✅ Smooth animations

## How to Use

1. Open `index.html` in your browser
2. Type a task in the input field
3. Click "Add" or press Enter to add a task
4. Click the checkbox to mark a task as complete
5. Click "Delete" to remove a task
6. Use filter buttons to view different task categories
7. Click "Clear Completed" to remove all completed tasks

## Learning Concepts

- DOM manipulation
- Event listeners
- Array methods (map, filter)
- localStorage API
- State management
- Form handling
- ES6 features
