# 06 - Expense Tracker

Ek comprehensive expense tracking application jiss mein aap apni income aur expenses ko manage kar sakte hain.

## Features

✅ Income aur Expense transactions add karein
✅ Transactions ko type ke alavas categorize karein
✅ Real-time balance calculation
✅ Transaction history with dates
✅ Individual transactions delete karein
✅ Tamam transactions clear karein
✅ Persistent storage (localStorage)
✅ Balance breakdown (Total, Income, Expenses)
✅ Responsive design

## Kaise Use Karein

1. `index.html` ko apne browser mein open karein
2. Transaction ki description likhin
3. Amount darj karein
4. Type select karein (Income ya Expense)
5. "Add Transaction" button par click karein
6. Transaction history dekhin running balance ke saath
7. Individual transactions delete karein ya tamam clear karein

## Learning Concepts

- Array manipulation
- localStorage API
- Form validation
- DOM manipulation
- Mathematical calculations
- Date handling
- Event listeners
