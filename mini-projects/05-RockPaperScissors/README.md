# 05 - Rock Paper Scissors

A classic Rock Paper Scissors game where you play against the computer.

## Features

✅ Play against computer
✅ Score tracking
✅ Three choices (Rock, Paper, Scissors)
✅ Result display with emojis
✅ Reset game button
✅ Responsive design
✅ Smooth animations

## How to Play

1. Open `index.html` in your browser
2. Click on Rock (✊), Paper (✋), or Scissors (✌️)
3. The computer makes its choice
4. Win/lose/tie is determined
5. Score is updated
6. Click "Reset Game" to start over

## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Learning Concepts

- Random number generation
- Conditional logic
- Game logic
- Score tracking
- DOM manipulation
- Event handling
