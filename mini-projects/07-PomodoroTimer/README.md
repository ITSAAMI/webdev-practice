# 07 - Pomodoro Timer

Pomodoro Technique par based ek productivity timer jis mein work aur break intervals hain.

## Features

✅ 25-minute work sessions
✅ 5-minute break sessions
✅ Customizable work aur break durations
✅ Start, pause, aur reset functionality
✅ Visual progress indicator (SVG circle)
✅ Audio notification on timer completion
✅ Automatic switch between work aur break
✅ Real-time display
✅ Responsive design

## Kaise Use Karein

1. `index.html` ko apne browser mein open karein
2. (Optional) Work aur break durations ko adjust karein
3. "Start" button par click karein timer shuru karne ke liye
4. "Pause" button par click karein rok karne ke liye
5. "Reset" button par click karein timer reset karne ke liye
6. Timer automatically work aur break ke beech switch hota hai

## Learning Concepts

- setInterval aur clearInterval
- SVG manipulation
- Audio API
- Time calculations
- State management
- DOM manipulation
- Event listeners
