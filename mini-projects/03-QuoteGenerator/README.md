# 03 - Random Quote Generator

A beautiful quote generator application that displays random quotes with the ability to copy and share on social media.

## Features

✅ Display random quotes
✅ Copy quotes to clipboard
✅ Share on Twitter
✅ Smooth animations
✅ Responsive design
✅ Beautiful UI with blue accents
✅ Quote author attribution

## How to Use

1. Open `index.html` in your browser
2. Click "New Quote" to display a random quote
3. Click "Copy" to copy the quote to your clipboard
4. Click "Share on Twitter" to share the quote on Twitter

## Learning Concepts

- Array manipulation
- Random selection
- Clipboard API
- Event listeners
- DOM manipulation
- CSS animations
- Window object (open)
