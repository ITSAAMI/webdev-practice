# JavaScript Mini Projects

This folder contains a collection of JavaScript mini projects for learning and teaching purposes.

## Project List

### Beginner Level
1. **Calculator** - Basic arithmetic operations
2. **To-Do List** - Add, delete, and mark tasks as complete
3. **Random Quote Generator** - Display random quotes
4. **Weather App** - Fetch weather data from an API
5. **Rock Paper Scissors** - Interactive game against the computer

### Intermediate Level
6. **Expense Tracker** - Track income and expenses with visualization
7. **Pomodoro Timer** - Productivity timer with notifications
8. **Movie Search App** - Search movies using an API
9. **Notes App** - Create, edit, and delete notes with local storage
10. **Password Generator** - Generate secure passwords with customizable options

### Advanced Level
11. **Chat Application** - Real-time messaging with WebSocket
12. **E-Commerce Cart** - Shopping cart with product filtering
13. **Kanban Board** - Drag and drop task management
14. **Music Player** - Play, pause, skip with playlist management
15. **Real-Time Collaboration Tool** - Collaborative document editing

## Getting Started

Each project folder contains:
- `index.html` - HTML structure
- `style.css` - Styling
- `script.js` - JavaScript logic
- `README.md` - Project documentation and instructions

## How to Use

1. Navigate to the project folder
2. Open `index.html` in your browser
3. Follow the project-specific README for detailed instructions
