1. <img> Tag — Image Dikhana

<img> tag se hum page par images dikhate hain. Yeh self-closing tag hai — isay close nahi karna.

📝 Basic Syntax:

<img src="image-path" alt="description">

Important Attributes:

AttributePurposeZaroori?
srcImage ka path ya URL✅ Haan
altImage na load ho toh alternative text✅ Haan (accessibility)
widthWidth pixels meinOptional
heightHeight pixels meinOptional
titleHover par tooltipOptional

2. Online vs Local Images

📝 Online Image (URL se):

<img src="https://via.placeholder.com/300x200" alt="Placeholder Image">

📝 Local Image (file se):

<!-- Same folder mein -->
<img src="photo.jpg" alt="Meri Photo">

<!-- Sub-folder mein -->
<img src="images/photo.jpg" alt="Meri Photo">

<!-- Parent folder mein -->
<img src="../photo.jpg" alt="Meri Photo">

3. Alt Attribute — Kyun Zaroori Hai?

alt attribute 3 kaam karta hai:

  • 🖥️ Image na load ho: Alt text dikhta hai image ki jagah
  • ♿ Screen Readers: Blind users ke liye screen reader alt text padhta hai
  • 🔍 SEO: Google bhi alt text padhta hai image ko samajhne ke liye

📝 Good vs Bad Alt Text:

<!-- ❌ BAD — empty ya generic -->
<img src="photo.jpg" alt="">
<img src="photo.jpg" alt="image">

<!-- ✅ GOOD — descriptive -->
<img src="photo.jpg" alt="Code Academy Nowshera ka classroom, students coding seekh rahe hain">

⚠️ Rule: alt attribute HAMESHA likhein — chahe khali (alt="") hi kyun na ho (decorative images ke liye).

4. Width aur Height Set Karna

📝 HTML Attributes Se:

<img src="photo.jpg" alt="Photo" width="300" height="200">

📝 CSS Se (Better Way):

<img src="photo.jpg" alt="Photo" style="width: 300px; height: auto;">

💡 Tip: Sirf width set karein aur height ko auto rakhein — image ka ratio maintain rahega.

6. Responsive Images

Image ko responsive banane ke liye CSS use karein:

<img src="photo.jpg" alt="Responsive" style="max-width: 100%; height: auto;">

Yeh image apne container se badi nahi hogi, aur chhote screens par automatically chhoti ho jayegi.

7. <figure> aur <figcaption>

Image ko caption ke saath dikhane ke liye <figure> aur <figcaption> use karein:

📝 Code:

<figure>
  <img src="nowshera.jpg" alt="Nowshera City View" style="max-width:100%;">
  <figcaption>Nowshera City ka khubsurat manzar</figcaption>
</figure>
