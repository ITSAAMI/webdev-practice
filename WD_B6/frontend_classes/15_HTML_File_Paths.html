1. File Paths Kya Hain?

File path se browser ko batate hain ke koi file (image, CSS, JS, page) kahan rakhi hai. Yeh 4 jagah use hote hain:

<img src="FILE_PATH">           <!-- Images -->
<a href="FILE_PATH">            <!-- Links -->
<link href="FILE_PATH">         <!-- CSS files -->
<script src="FILE_PATH">        <!-- JavaScript files -->

3. Absolute Path — Full URL

Absolute path mein complete URL hota hai — external websites ke liye:

<!-- External website ki image -->
<img src="https://www.example.com/images/logo.png" alt="Logo">

<!-- External website ka link -->
<a href="https://www.google.com">Google</a>

<!-- External CSS -->
<link href="https://cdn.example.com/style.css" rel="stylesheet">

📌 Absolute path external resources (CDN, APIs, doosri websites) ke liye use karein.

4. Relative Paths — Folder Structure Se

Relative path apni website ke files ke liye — current file ki position se dekhte hain:

PathMatlabExample (from index.html)
file.jpgSame folder mein<img src="logo.png">
folder/file.jpgSub-folder mein<img src="images/logo.png">
../file.jpgParent folder mein (ek upar)<img src="../logo.png">
../../file.jpg2 folders upar<img src="../../logo.png">
./file.jpgCurrent folder (same as file.jpg)<img src="./logo.png">

📝 Practical Examples (from index.html):

<!-- Same folder ki file -->
<a href="about.html">About Page</a>

<!-- Sub-folder ki file -->
<img src="images/logo.png" alt="Logo">
<link href="css/style.css" rel="stylesheet">
<script src="js/script.js"></script>

<!-- Nested sub-folder -->
<img src="images/gallery/pic1.jpg" alt="Gallery Pic">

<!-- Sub-folder ki HTML file -->
<a href="pages/services.html">Services</a>

5. ../ — Parent Folder Mein Jaana

Agar aap pages/services.html mein hain aur root ki file chahiye:

<!-- pages/services.html SE -->

<!-- Root folder ki file (ek upar) -->
<a href="../index.html">Home</a>

<!-- Root ke images folder ki file -->
<img src="../images/logo.png" alt="Logo">

<!-- Root ke css folder ki file -->
<link href="../css/style.css" rel="stylesheet">

💡 ../ = ek folder upar jao. ../../ = do folders upar jao. Har ../ ek level upar le jata hai.

src="logo .png"File naam mein spaceSpaces avoid karein ya %20 use karein

⚠️ Golden Rules:
✅ Forward slash / use karein (backslash nahi)
✅ Lowercase file names rakhein
✅ Spaces ki jagah dash - use karein
✅ Desktop paths kabhi na use karein

7. Folder Structure Best Practices

  • 📁 Images ko images/ folder mein rakhein
  • 📁 CSS ko css/ folder mein rakhein
  • 📁 JavaScript ko js/ folder mein rakhein
  • 📄 Main page ka naam index.html rakhein (server default)
  • 🔤 Sab file/folder names lowercase mein rakhein
  • ➖ Spaces ki jagah hyphens use karein: my-page.html
  • 📂 Related files ko folders mein organize karein

📝 Practice Tasks

  • Task 1: Ek multi-page website banayein (index.html, about.html, contact.html). Sab pages ek doosre se linked hon using relative paths. Images ko images/ folder mein rakhein.
  • Task 2: Sub-folder pages/ mein ek page banayein jo parent folder (../) ki images aur CSS use kare.
  • Task 3: Apne project ki folder structure ka diagram banayein (text se) aur har file ka path likhein.
