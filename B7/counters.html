What is RAM?

RAM stands for Random Access Memory. It is the short-term memory of a computer, where data is stored temporarily while the computer is running. Think of RAM like a whiteboard in a classroom — you can write on it quickly, erase it, and write again, but once you wipe it, it’s gone.

Types of RAM

DRAM (Dynamic RAM)

  • Needs to be refreshed many times per second.
  • Slower but cheaper.
  • Like a chalkboard — you can write, but you need to keep re-writing before it fades.
  • Sub-types of DRAM

    • SDRAM (Synchronous DRAM)

      Works in sync with the computer’s clock speed.

      Example: Like walking in step with your friend — both move together in rhythm.

    • DDR SDRAM (Double Data Rate SDRAM)

      Transfers data twice per clock cycle.

      Example: Like carrying two shopping bags instead of one per trip.

    • DDR2, DDR3, DDR4, DDR5

      Each new generation is faster and uses less power.

      Example: Upgrading from a bicycle to a motorcycle to a car — each is faster and more efficient.

    SRAM (Static RAM)

    • Does not need frequent refreshing.
    • Faster but more expensive.
    • Like sticky notes — stays there until you remove it.

    Why is RAM Important?

    • More RAM = more programs open at the same time without slowing down.
    • Example: Like having a bigger table to work on — you can keep more books open at once.
