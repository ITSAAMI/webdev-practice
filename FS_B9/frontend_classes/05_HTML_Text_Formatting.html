1. Bold Text — <b> aur <strong>

Dono tags text ko bold banate hain, lekin meaning alag hai:

TagPurposeSEO / Screen Reader
<b>Sirf visually bold — koi extra meaning nahi❌ Koi importance nahi
<strong>Important content — bold + semantic meaning✅ Screen reader emphasis deta hai

📝 Code:

<p><b>Yeh sirf bold hai</b> — visual formatting.</p>
<p><strong>Yeh important hai</strong> — semantic + bold.</p>

🖥️ Output:

Yeh sirf bold hai — visual formatting.

Yeh important hai — semantic + bold.

💡 Best Practice: Hamesha <strong> use karein jab content important ho. <b> sirf styling ke liye.

2. Italic Text — <i> aur <em>

TagPurposeSemantic?
<i>Visually italic — terms, titles ke liye❌
<em>Emphasized text — stress/emphasis ke liye✅

📝 Code:

<p><i>Yeh italic text hai</i> — jaise book ka naam.</p>
<p><em>Yeh emphasized hai</em> — isko zor se padhein.</p>

🖥️ Output:

Yeh italic text hai — jaise book ka naam.

Yeh emphasized hai — isko zor se padhein.

3. Underline, Strikethrough, aur Highlight

📝 Code:

<!-- Underline -->
<p><u>Yeh underlined text hai</u></p>

<!-- Strikethrough — purani ya galat info ke liye -->
<p>Price: <s>Rs. 5000</s> Rs. 3000</p>
<p><del>Yeh delete hua hai</del> <ins>Yeh naya hai</ins></p>

<!-- Highlighted text -->
<p>Yeh <mark>highlighted</mark> text hai.</p>

🖥️ Output:

Yeh underlined text hai

Price: Rs. 5000 Rs. 3000

Yeh delete hua hai Yeh naya hai

Yeh highlighted text hai.

Tags Ka Comparison:

TagEffectUse Case
<u>UnderlineMisspelled words, annotations
<s>StrikethroughPurani prices, outdated info
<del>Deleted text (semantic)Document revisions
<ins>Inserted text (underlined)Newly added content
<mark>Yellow highlightSearch results, important words

4. Small Text, Subscript & Superscript

📝 Code:

<!-- Small text -->
<p>Normal text <small>aur yeh chhota text hai</small></p>

<!-- Subscript — chemical formulas ke liye -->
<p>Pani ka formula: H<sub>2</sub>O</p>
<p>Carbon Dioxide: CO<sub>2</sub></p>

<!-- Superscript — math powers ke liye -->
<p>Area = r<sup>2</sup> × π</p>
<p>Einstein: E = mc<sup>2</sup></p>

🖥️ Output:

Normal text aur yeh chhota text hai

Pani ka formula: H2O

Carbon Dioxide: CO2

Area = r2 × π

Einstein: E = mc2

5. Abbreviation, Quotations & Citations

📝 Code:

<!-- Abbreviation — hover par full form dikhega -->
<p><abbr title="HyperText Markup Language">HTML</abbr> seekhein.</p>

<!-- Block Quote — bada quote, indent hota hai -->
<blockquote>
  "Ilm haasil karo chahe China jaana pade."
</blockquote>

<!-- Inline Quote — chhota quote, quotation marks aate hain -->
<p>Iqbal ne kaha: <q>Khudi ko kar buland itna</q></p>

<!-- Citation — book, movie, article ka naam -->
<p>Meri favourite kitaab <cite>Bang-e-Dra</cite> hai.</p>

<!-- Inline Code -->
<p>HTML mein heading ke liye <code>&lt;h1&gt;</code> tag use karein.</p>

🖥️ Output:

HTML seekhein. (hover karein)

"Ilm haasil karo chahe China jaana pade."

Iqbal ne kaha: Khudi ko kar buland itna

Meri favourite kitaab Bang-e-Dra hai.

HTML mein heading ke liye <h1> tag use karein.

📝 Practice Tasks

  • Task 1: Ek product card banayein jisme product ka naam <strong> mein ho, purani price <del> mein ho, aur nayi price <ins> mein ho. Description mein <mark> se key features highlight karein.
  • Task 2: Ek science article likhein jisme chemical formulas (<sub>), math equations (<sup>), aur scientific terms (<abbr>) use hon.
  • Task 3: Ek famous quotes page banayein — 5 quotes <blockquote> mein likhein, har quote ke neeche author ka naam <cite> mein ho.
