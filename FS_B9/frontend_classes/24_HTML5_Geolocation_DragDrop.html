1. Geolocation API — User Ki Location

Geolocation API se aap user ki current location (Latitude aur Longitude) pata kar sakte hain. Browser pehle user se permission mangta hai.

<script>
function getLocation() {
  if (navigator.geolocation) {
    // Permission mago aur location get karo
    navigator.geolocation.getCurrentPosition(showPosition, showError);
  } else {
    alert("Aapka browser Geolocation support nahi karta.");
  }
}

function showPosition(position) {
  let lat = position.coords.latitude;
  let lon = position.coords.longitude;
  document.getElementById("locResult").innerText = "Latitude: " + lat + ", Longitude: " + lon;
}

function showError(error) {
  // Error handling (User ne allow nahi kiya, ya location off hai)
}
</script>

2. Drag and Drop API

HTML5 mein aap kisi bhi element ko "draggable" bana sakte hain aur JavaScript ke zariye events (dragstart, dragover, drop) handle kar sakte hain.

Basic Steps:

  1. Element ko draggable="true" dein.
  2. ondragstart event par data set karein.
  3. Drop zone par ondragover ko allow karein (preventDefault).
  4. Drop zone par ondrop event handle karein.
<!-- Draggable Element -->
<div id="dragItem" draggable="true" ondragstart="drag(event)">Pakdo Mujhe!</div>

<!-- Drop Zone -->
<div id="dropZone" ondrop="drop(event)" ondragover="allowDrop(event)"></div>

<script>
  function allowDrop(ev) {
    ev.preventDefault();  // Drop ko allow karta hai
  }

  function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id); // ID save karo
  }

  function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");    // ID retrieve karo
    ev.target.appendChild(document.getElementById(data)); // Append kardo
  }
</script>
