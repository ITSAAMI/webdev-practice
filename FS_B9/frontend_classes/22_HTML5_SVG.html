1. SVG Kya Hai? (Scalable Vector Graphics)

SVG XML-based vector graphics hain. Inki khaasiyat yeh hai ke inhein kitna bhi zoom kar lo, inki quality kharab nahi hoti (pixelate nahi hote).

SVG (Vector)Canvas (Raster)
Zoom karne par quality same rehti haiZoom karne par pixelate/blur ho jate hain
XML syntax (tags hote hain)JavaScript se drawing hoti hai
CSS se style ho sakte hainJavaScript se style hote hain
Icons aur logos ke liye best haiGames aur complex animations ke liye best hai

2. Basic SVG Shapes

Rectangle <rect>

<svg width="200" height="100">
  <rect width="150" height="80" x="25" y="10" fill="#e94560" rx="10" />
</svg>

rx se corners round hote hain

Circle <circle>

<svg width="150" height="150">
  <circle cx="75" cy="75" r="50" fill="#0f3460" stroke="#00d2ff" stroke-width="4" />
</svg>

Polygon <polygon>

<svg width="150" height="150">
  <polygon points="75,10 140,140 10,140" fill="#00d2ff" />
</svg>

3. Complex Shapes & Text

Path <path>

<path> sabse powerful shape hai. d attribute mein path data hota hai (M = move to, L = line to, Q/C = curve, Z = close).

<svg width="150" height="150">
  <path d="M 10 75 Q 75 10 140 75 T 140 140" fill="none" stroke="#e94560" stroke-width="5" />
</svg>

Text <text>

<svg width="250" height="50">
  <text x="10" y="35" fill="#e94560" font-size="24" font-weight="bold">Code Academy</text>
</svg>

4. Styling with CSS

Inline SVG tags ko direct CSS se target kiya ja sakta hai:

<style>
  .my-circle { fill: #0f3460; transition: fill 0.3s; }
  .my-circle:hover { fill: #e94560; }
</style>

<svg width="100" height="100">
  <circle class="my-circle" cx="50" cy="50" r="40" />
</svg>

💡 Pro Tip: SVG mein color ya background-color nahi hota, balki fill aur stroke properties use hoti hain.

📝 Practice Tasks

  • Task 1: SVG shapes use karke ek simple hut (ghar) draw karein — rectangle, polygon (roof), aur lines/rectangles (windows/doors).
  • Task 2: SVG se ek colorful button banayein jisme text aur ek rounded rectangle ho. Hover par CSS se iska fill color change karein.
  • Task 3: <path> use karke ek abstract curvy design banayein aur usko stroke styling dein.
