1. HTML Headings Kya Hain?

Headings se hum page ke titles aur sections define karte hain. HTML mein 6 levels ki headings hain:

<h1>Heading 1 — Sabse Bada</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6 — Sabse Chhota</h6>

🖥️ Browser Output:

Heading 1 — Sabse Bada

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6 — Sabse Chhota

2. Har Heading Ka Size Aur Purpose

Tag Default Size Use Case
<h1> 2em (32px) Page ka main title — sirf EK baar use karein
<h2> 1.5em (24px) Major sections ke titles
<h3> 1.17em (18.72px) Sub-sections
<h4> 1em (16px) Sub-sub-sections
<h5> 0.83em (13.28px) Detailed sections
<h6> 0.67em (10.72px) Sabse neeche level ki heading — kam use hota hai

3. SEO Mein Headings Ki Importance

SEO (Search Engine Optimization) mein headings bohat important hain:

  • 🔍 Google page ki <h1> padhta hai taake samjhe page kis baare mein hai
  • 📊 Search results mein headings ka text dikhta hai
  • 📱 Screen readers headings use karke page navigate karte hain
  • 🏗️ Headings se page ka outline/structure banta hai

💡 SEO Rule: Ek page mein sirf EK <h1> hona chahiye — yeh page ka main topic batata hai.

4. Heading Hierarchy — Sahi Tartib

Headings ko hamesha sahi order mein use karein — h1 ke baad h2, h2 ke baad h3:

✅ Sahi Hierarchy:

<h1>Pakistan Ka Geography</h1>

  <h2>Provinces</h2>
    <h3>KPK</h3>
    <h3>Punjab</h3>
    <h3>Sindh</h3>

  <h2>Major Cities</h2>
    <h3>Islamabad</h3>
    <h3>Karachi</h3>

❌ Galat Hierarchy (Skip karna):

<h1>Main Title</h1>
<h4>Seedha h4 par aa gaye — h2, h3 skip ho gaye!</h4>

⚠️ Galti na karein: Sirf size change karne ke liye heading level mat change karein. Size ke liye CSS use karein!

5. Headings Ke Saath Paragraphs

Real websites mein heading ke baad related paragraph aata hai:

📝 Code:

<h1>Code Academy Nowshera</h1>
<p>Hum aik modern coding academy hain jo web development sikhate hain.</p>

<h2>Hamare Courses</h2>
<p>HTML, CSS, JavaScript, React, Node.js aur bohat kuch.</p>

<h2>Kyun Chunein Humein?</h2>
<p>Experienced instructors, hands-on projects, aur career support.</p>

🖥️ Browser Output:

Code Academy Nowshera

Hum aik modern coding academy hain jo web development sikhate hain.

Hamare Courses

HTML, CSS, JavaScript, React, Node.js aur bohat kuch.

Kyun Chunein Humein?

Experienced instructors, hands-on projects, aur career support.

6. CSS Se Heading Customize Karna

Heading ka size, color, font change karne ke liye CSS use karein:

<h2 style="color: #e94560; font-size: 28px; font-family: Arial;">
  Styled Heading
</h2>

<h3 style="color: #00d2ff; text-align: center; text-transform: uppercase;">
  Centered Uppercase Heading
</h3>

🖥️ Output:

Styled Heading

Centered Uppercase Heading

7. Common Mistakes ❌

MistakeKyun Galat HaiSahi Tareeqa
Multiple <h1> tags SEO mein confuse karta hai Sirf EK <h1> per page
h1 ke baad seedha h4 use karna Hierarchy toot jati hai h1 → h2 → h3 → h4 order follow karein
Size ke liye heading level change karna Semantic meaning khatam ho jati hai CSS se size change karein
Paragraph text ko heading mein likhna Heading bold/big hota hai but yeh galat use hai Normal text ke liye <p> use karein

📝 Practice Tasks

  • Task 1: Ek "My School" page banayein jisme h1 se h6 tak tamam headings use hon — har heading ke baad ek paragraph ho.
  • Task 2: Apne favourite subject ki article likhein with proper heading hierarchy (h1 → h2 → h3). Kam az kam 3 sections hon.
  • Task 3: Headings ko CSS se style karein — different colors, sizes, aur fonts use karein.
