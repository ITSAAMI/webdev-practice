1. HTML Document Ka Structure

Har HTML page ka ek fixed structure hota hai — jaise ghar banane ke liye neev, deewarein aur chhat zaroori hain, waise hi HTML page ke liye kuch zaroori tags hain:

📝 Complete HTML Boilerplate:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Ka Title</title>
</head>
<body>
  <h1>Yeh visible content hai</h1>
  <p>Browser mein yeh dikhega.</p>
</body>
</html>

💡 VS Code Shortcut: Nayi HTML file mein ! type karke Tab ya Enter dabayein — poora boilerplate automatic aa jayega!

2. <!DOCTYPE html> — Document Type Declaration

Yeh line browser ko batati hai ke yeh document HTML5 standard mein likha gaya hai.

<!DOCTYPE html>

Key Points:

  • Yeh tag sabse pehle likhna hai — <html> se bhi pehle
  • Yeh ek declaration hai, tag nahi — isay close nahi karte
  • Case-insensitive hai — <!doctype html> bhi chalega
  • Agar nahi likhein toh browser "quirks mode" mein chala jayega — page sahi nahi dikhega

3. <html> — Root Element

Yeh root element hai — poora HTML document iske andar wrap hota hai.

<html lang="en">
  <!-- Sab kuch yahan andar aayega -->
</html>

lang Attribute:

  • lang="en" — English content ke liye
  • lang="ur" — Urdu content ke liye
  • Search engines aur screen readers ko content ki language batata hai

4. <head> — Metadata Area (Invisible Section)

Head section mein woh information hoti hai jo browser mein nahi dikhti lekin important hai:

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mera Page</title>
  <link rel="stylesheet" href="style.css">
  <script src="script.js"></script>
</head>

Head Mein Kya Kya Aa Sakta Hai:

TagKya Karta Hai
<title>Browser tab ka title set karta hai
<meta charset>Character encoding batata hai (UTF-8 sab characters support karta hai)
<meta viewport>Mobile responsiveness ke liye zaroori hai
<link>External CSS file connect karta hai
<script>JavaScript file connect karta hai
<style>Internal CSS likhne ke liye

5. <body> — Visible Content Area

Jitna bhi content aapko screen par dikhana hai — headings, paragraphs, images, buttons — sab <body> ke andar likhein:

<body>
  <h1>Welcome to My Website</h1>
  <p>Yeh mera content hai.</p>
  <img src="photo.jpg" alt="Meri Photo">
</body>

📌 Rule: Ek HTML document mein sirf ek <body> tag hota hai!

6. <title> — Browser Tab Ka Title

Title tag se jo text likhein woh browser ke tab mein dikhta hai:

<title>Code Academy Nowshera - Home</title>

Title Ki Importance:

  • Browser tab mein dikhta hai
  • Google search results mein page ka title banta hai (SEO)
  • Bookmarks mein save hone wala naam yahi hota hai

7. Nesting — Tags Ke Andar Tags

Nesting ka matlab hai ek tag ke andar doosra tag rakhna. HTML ek tree structure follow karta hai:

<html>
  ├── <head>
  │   ├── <meta>
  │   └── <title>
  └── <body>
      ├── <h1>
      └── <p>

✅ Sahi Nesting:

<p><strong>Yeh bold text hai</strong></p>

❌ Galat Nesting:

<p><strong>Yeh galat hai</p></strong>

⚠️ Jo tag pehle khulta hai, woh baad mein band hota hai — LIFO (Last In, First Out) rule.

8. Indentation — Code Ko Readable Banayein

Indentation matlab har nested tag ko 2 ya 4 spaces andar likhna. Browser ko fark nahi parta, lekin:

❌ Bina Indentation (Mushkil padhna):

<html><head><title>Test</title></head><body><h1>Hello</h1></body></html>

✅ Indentation Ke Saath (Easy padhna):

<html>
  <head>
    <title>Test</title>
  </head>
  <body>
    <h1>Hello</h1>
  </body>
</html>

💡 VS Code Shortcut: Code select karke Shift + Alt + F dabayein — auto-format ho jayega!
